Projects

HTTP Server

A multithreaed HTTP Server that can handle cURL requests including GET, PUT, and APPEND. It also allows full customization on how many threads to open and which port to use. The program is written completely in C using the pthread library and can be found on my Github page.

cURL replica

A Python program designed to replicate the basic cURL functions. Similar to the HTTP Server above, this parses the request and return the status code and the content of the page. However, this is simplified and a specific port is designed to not work. The program also creates a csv file to log the entries of each cURL requests. You can also track these requests via WireShark and see each individual requests and responses.
Link to the repo.

Traveling Salesman Problem

The Traveling Salesman problem is a well known problem that has not been solved yet. This program is an attempt to solve the problem. It can handle up to 26 vertices and will find the shortest path between 2 vertices using Depth First Search. It also supports piping the output to a file. If not specified, then it will output to stdout.
Link to the repo.
